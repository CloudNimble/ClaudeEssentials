## Summary

Represents the input received by a PermissionRequest hook.
            This hook runs when a permission dialog is shown and can automatically allow or deny permissions.

## Type Parameters

- `TToolInput` - The type representing the tool's input parameters.
            Use a specific tool input class or [Object](https://learn.microsoft.com/dotnet/api/system.object) for dynamic inputs.
